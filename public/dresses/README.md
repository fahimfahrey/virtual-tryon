# Dresses Folder

Place your dress PNG images here with these exact filenames:

- dress1.png — Red Evening Gown
- dress2.png — Blue Cocktail Dress
- dress3.png — Floral Summer Dress
- dress4.png — Black Formal Dress
- dress5.png — White Bridal Gown

Images should be:

- PNG format with transparent background (preferred) or white background
- Recommended size: 512x768px or similar portrait ratio
- Full dress visible from top to bottom

To add more dresses, add the PNG here and update the DRESSES array in app/page.js.
